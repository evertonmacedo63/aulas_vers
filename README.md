## 🧭 Sobre este repositório

Este espaço foi criado para organizar e distribuir **artefatos orientativos**, materiais de estudo e exemplos utilizados nas aulas do **3ºF – Leos 2026**.  
Além disso, ele serve como **modelo de portfólio** para que os alunos aprendam a montar seus próprios repositórios no GitHub de forma clara, bonita e profissional.
